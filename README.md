# Tmux Statusline

Personalized Tmux statusline based on [wfxr/tmux-power](https://github.com/wfxr/tmux-power/tree/master).

## Installation

```
set -g @plugin 'jpinilloslr/tmux-statusline'
```

## Optional extensions

```
set -g @plugin 'olimorris/tmux-pomodoro-plus'
```
